In diesem Ordner befinden sich die Programme für die Erstellung der Bilder und Videos zur Projektion von Quadrat, Würfel und Tesserakt.

Diese Programme können wie jedes Python-Programm gestartet werden.
Dafür muss Python installiert sein: https://www.python.org

Eventuell muss matplotlib zusätzlich installiert werden.
Die Anleitung dafür ist unter folgendem Link zu finden:
https://matplotlib.org/stable/users/installing/index.html

Bei der Erstellung wurden folgende Versionen verwendet:
Python Version: 3.9.13
Numpy Version: 1.21.5
Matplotlib Version: 3.5.2

Die Programme:
Tesseract.py
Cube.py
SquareCProjection.py
SquarePProjection.py
sind Hilfsprogramme.
Wenn sie gestartet werden passiert nichts.
In diesen Programmen sind die Funktionen für Projektion, Rotation, etc.
Aus Platzgründen werden diese Programme von den anderen Programmen, die die Animationen erstellen, importiert.
In diesen 4 Programmen befindet sich auch Beispiele für das Erstellen eines Bildes, sie wurden allerdings auskommentiert.
